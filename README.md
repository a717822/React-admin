## 如何使用
``` bash
# 安装依赖
npm install

# 启动服务在localhost:3000
npm start
```

## 依赖介绍
``` bash
1、ant design 
2、babel-polyfill 解决ES5、ES6对IE浏览器的解决方案
3、react-router/react-router-dom React路由
4、wangeditor3 富文本编辑器
```
